Introdução ao HTML Voltar

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. É uma linguagem usada para criar e estruturar páginas web, que podem ser visualizadas em um navegador. O HTML é composto por elementos chamados tags, que definem o conteúdo e o layout da página. Cada tag tem um nome e pode ter atributos, que são informações adicionais sobre o elemento. As tags são escritas entre sinais de menor e maior (< e >), e geralmente vêm em pares: uma tag de abertura e uma tag de fechamento. Por exemplo: <p>Este é um parágrafo</p>.


Introdução à programação WEB Voltar

A programação web é o processo de criar e manter sites e aplicações que funcionam na internet. Existem dois tipos de programação web: front-end e back-end. O front-end é a parte que o usuário vê e interage, como o design, as cores, as imagens, os botões, os formulários etc. O back-end é a parte que fica nos servidores, e é responsável por processar os dados, fazer cálculos, conectar com bancos de dados, enviar e-mails etc. O HTML é uma das linguagens usadas para o desenvolvimento do front-end, junto com outras como CSS e JavaScript.


A estrutura básica de uma página HTML Voltar

Uma página HTML tem uma estrutura padrão, que consiste em três partes principais: a declaração do tipo de documento (DOCTYPE), a tag <html> e a tag <head> e a tag <body>.

Entendendo a estrutura básica de uma página HTML Voltar

A declaração do tipo de documento (DOCTYPE) é a primeira linha de uma página HTML, e serve para informar ao navegador qual é a versão do HTML que está sendo usada. A versão mais atual é o HTML5, e a declaração é simplesmente: <!DOCTYPE html>.

A tag <html> é a raiz de uma página HTML, e envolve todo o conteúdo da página. Ela pode ter um atributo chamado lang, que indica o idioma da página. Por exemplo: <html lang="pt-br">.

A tag <head> é a cabeça da página HTML, e contém informações que não são mostradas diretamente na página, mas que são importantes para o funcionamento ou a descrição da página. Por exemplo: o título da página, as meta tags, os links para arquivos externos etc.

A tag <title> é uma das tags que ficam dentro da tag <head>, e define o título da página, que aparece na aba ou na barra de título do navegador. Por exemplo: <title>Introdução ao HTML</title>.

A tag <body> é o corpo da página HTML, e contém o conteúdo que é mostrado na página, como textos, imagens, links, listas, tabelas, formulários etc.

Um exemplo de uma página HTML básica é:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
   <title>Introdução ao HTML</title>
</head>
<body>
    <h1>Introdução ao HTML</h1>
    <p>HTML é a linguagem usada para criar páginas web.</p>
</body>
</html>

Falando sobre Tags Voltar

O que são e como funcionam Voltar

As tags são os elementos que compõem uma página HTML, e definem o tipo, o conteúdo e o formato de cada parte da página. Cada tag tem um nome, que é escrito entre sinais de menor e maior (< e >). Algumas tags têm uma barra (/) antes do nome, e são chamadas de tags de fechamento. Elas indicam o fim de um elemento. Por exemplo: </p> é a tag de fechamento da tag <p>. Outras tagsnão têm uma tag de fechamento, e são chamadas de tags não contêineres. Elas são usadas para inserir elementos que não têm conteúdo, como imagens,linhas horizontais, quebras de linha etc. Por exemplo: <img> é a tag usada para inserir uma imagem.

Tipos de tags: contêiner e não contêiner. Voltar

As tags contêineres são aquelas que têm uma tag de abertura e uma tag de fechamento, e que envolvem um conteúdo. Por exemplo:
<p>Este é um parágrafo</p>. As tags não contêineres são aquelas que não têm uma tag de fechamento, e que não envolvem um conteúdo. Por exemplo: <src="imagem.jpg">. Alguns exemplos de tags contêineres são:

  • <h1> a <h6>: são usadas para definir títulos e subtítulos na página. O número indica o nível de importância do título, sendo <h1>; o mais importante e <h6> o menos importante. Por exemplo: <h1>Introdução ao HTML</h1> é o título principal da página.

  • <p>: é usada para definir um parágrafo na página. Por exemplo: <p>HTML é a linguagem usada para criar páginas web.</p> é um parágrafo.

  • <div>: é usada para definir uma divisão ou uma seção na página. Por exemplo: <div id="menu">...</div> é uma divisão que contém o menu da página.

  • <span>: é usada para definir uma parte de um texto na página. Por exemplo: <p>Este é um texto com uma <span style="color: red;">palavra</span> em vermelho.</p> é um parágrafo com uma parte em vermelho.

    Este é um texto com uma palavra em vermelho.

  • <a>: é usada para definir um link na página. Por exemplo: <a href="https://www.dio.me">DIO.me</a> é um link para o site da DIO.me.

    DIO.me

Alguns exemplos de tags não contêineres são:

  • <img>: é usada para inserir uma imagem na página. Ela tem um atributo chamado src, que indica o endereço da imagem. Por exemplo: <img src="dio.png"> é uma imagem com o endereço “dio.png”.

  • <br>: é usada para inserir uma quebra de linha na página. Por exemplo: <p>Este é um texto<br>com duas linhas.</p> é um parágrafo com duas linhas.

    Este é um texto
    com duas linhas.

  • <hr>: é usada para inserir uma linha horizontal na página. Por exemplo: <hr> é uma linha horizontal.

  • <input>: é usada para inserir um campo de entrada na página. Ela tem um atributo chamado type, que indica o tipo de entrada. Por exemplo: <input type="text"> é um campo de texto.


Atributo das tags Voltar

O que são e como funcionam Voltar

Os atributos são informações adicionais que podem ser colocadas nas tags, para modificar ou especificar o comportamento ou a aparência dos elementos. Cada atributo tem um nome e um valor, que são escritos dentro da tag, separados por um sinal de igual (=). O valor do atributo deve estar entre aspas duplas ("). Por exemplo: <p style="color: blue;">Este é um parágrafo azul</p> é um parágrafo com um atributo chamado style, que tem o valor “color: blue;”, que significa que a cor do texto é azul.

Este é um parágrafo azul

Tipos de atributos: genéricos e específicos Voltar

Os atributos podem ser classificados em dois tipos: genéricos e específicos. Os atributos genéricos são aqueles que podem ser usados em qualquer tag, e que servem para identificar, classificar ou estilizar os elementos. Os atributos específicos são aqueles que só podem ser usados em algumas tags, e que servem para definir características ou funções específicas dos elementos. Alguns exemplos de atributos genéricos são:

  • id: é usado para dar um identificador único a um elemento na página. Por exemplo: <div id="menu">...</div> é uma divisão que tem o identificador “menu”.

  • class: é usado para dar uma ou mais classes a um elemento na página. As classes servem para agrupar elementos que têm características em comum, e para facilitar a aplicação de estilos. Por exemplo: <p class="destaque">Este é um parágrafo destacado</p> é um parágrafo que tem a classe “destaque”.

  • style: é usado para dar um estilo a um elemento na página. O estilo é uma série de propriedades e valores que definem a aparência do elemento, como a cor, o tamanho, a fonte, o alinhamento, o espaçamento, o fundo, a borda etc. Por exemplo: <p style="color: blue; font-size: 20px; text-align: center;">Este é um parágrafo estilizado</p> é um parágrafo com um estilo que define a cor do texto como azul, o tamanho da fonte como 20 pixels, e o alinhamento do texto como centralizado.

    Este é um parágrafo estilizado

  • type: é usado para definir o tipo de um elemento na página. Ele pode ser usado em tags como <input>, <button>, <script>, <style> etc. Por exemplo: <input type="text"> é um campo de texto, <button type="submit"> é um botão de envio, <script type="text/javascript"> é um código JavaScript, <style type="text/css"> é um código CSS etc.

  • src: é usado para definir a fonte de um elemento na página. Ele pode ser usado em tags como <img>, <audio>, <video>, <iframe>, <script> etc. Por exemplo: <img src="imagem.jpg"> é uma imagem com o endereço “imagem.jpg”, <audio src="musica.mp3"> é um áudio com o endereço “musica.mp3”, <video src="video.mp4"> é um vídeo com o endereço “video.mp4”, <iframe src="https://www.dio.me"> é um quadro com o site da DIO.me, <script src="script.js"> é um código JavaScript externo etc.

    <iframe src="https://www.dio.me">

  • href: é usado para definir o endereço de um link na página. Ele pode ser usado na tag <a>, que é usada para criar links. Por exemplo:
    <a href="https://www.dio.me">DIO.me</a> é um link para o site da DIO.me.

  • target: é usado para definir o alvo de um link na página. Ele pode ser usado na tag <a>, e serve para indicar onde o link será aberto. Por exemplo: <a href="https://www.dio.me" target="_blank">DIO.me</a> é um link para o site da DIO.me que será aberto em uma nova aba.

  • title: é usado para definir o título de um elemento na página. Ele pode ser usado em várias tags, e serve para mostrar um texto quando o usuário passa o mouse sobre o elemento. Por exemplo: <a href="https://www.DIO.me" title="Este é o site da DIO.me">DIO.me</a> é um link para o site da DIO.me que mostra o texto “Este é o site da DIO.me” quando o usuário passa o mouse sobre ele.

    DIO.me


Textos e suas tags Voltar

Os textos são uma parte importante de uma página HTML, e podem ser formatados e organizados de diferentes formas, usando tags específicas. Algumas das tags mais usadas para trabalhar com textos são:

Tipografia HTML Voltar

A tipografia HTML é o conjunto de regras e técnicas que definem a aparência e o estilo dos textos em uma página HTML. A tipografia HTML envolve aspectos como o tipo, o tamanho, a cor, o alinhamento, o espaçamento, o destaque etc. A tipografia HTML pode ser definida usando o atributo style nas tags, ou usando uma linguagem chamada CSS, que é usada para estilizar as páginas web de forma mais avançada e organizada.

Principais tags e como o navegador se comporta com os seus usos Voltar

As principais tags usadas para definir textos em uma página HTML são:

  • <h1> a <h6>: são usadas para definir títulos e subtítulos na página. O navegador mostra os textos dessas tags com um tamanho e uma fonte diferentes, de acordo com o nível de importância do título. Por exemplo: <h1>Introdução ao HTML</h1> é o título principal da página, e o navegador mostra o texto com um tamanho maior e em negrito.

    Introdução ao HTML

  • <p>: é usada para definir um parágrafo na página. O navegador mostra os textos dessa tag com um tamanho e uma fonte padrão, e adiciona uma quebra de linha antes e depois do parágrafo. Por exemplo: <p>HTML é a linguagem usada para criar páginas web.</p> é um parágrafo, e o navegador mostra o texto com uma quebra de linha acima e abaixo dele.

    HTML é a linguagem usada para criar páginas web.

  • <blockquote>: é usada para definir uma citação na página. O navegador mostra os textos dessa tag com um recuo à esquerda e à direita, e com uma fonte diferente. Por exemplo: <blockquote>HTML é fácil de aprender.</blockquote> é uma citação, e o navegador mostra o texto com um recuo e uma fonte diferente.

    HTML é a linguagem usada para criar páginas web.
    HTML é fácil de aprender.

Tag <i> Voltar

A tag <i> é usada para definir um texto em itálico na página. O navegador mostra os textos dessa tag com uma fonte inclinada. Por exemplo: <p>Este é um texto com uma <i>palavra</i> em itálico.</p> é um parágrafo com uma parte em itálico.

Este é um texto com uma palavra em itálico.

Tag <strong> Voltar

A tag <strong> é usada para definir um texto em negrito na página. O navegador mostra os textos dessa tag com uma fonte mais grossa. Por exemplo: <p>Este é um texto com uma <strong>palavra</strong> em negrito.</p> é um parágrafo com uma parte em negrito.

Este é um texto com uma palavra em negrito.

Tag <u> Voltar

A tag <u> é usada para definir um texto sublinhado na página. O navegador mostra os textos dessa tag com uma linha abaixo da fonte. Por exemplo: <p>Este é um texto com uma <u>palavra</u> sublinhada.</p> é um parágrafo com uma parte sublinhada.

Este é um texto com uma palavra sublinhada.

Tag <mark> Voltar

A tag <mark> é usada para definir um texto marcado na página. O navegador mostra os textos dessa tag com uma cor de fundo diferente. Por exemplo: <p>Este é um texto com uma <mark>palavra</mark> marcada.</p> é um parágrafo com uma parte marcada.

Este é um texto com uma palavra marcada.

Tag <sup> Voltar

A tag <sup> é usada para definir um texto sobrescrito na página. O navegador mostra os textos dessa tag com uma fonte menor e acima da linha base. Por exemplo: <p>Este é um texto com uma <sup>palavra</sup> sobrescrita.</p> é um parágrafo com uma parte sobrescrita.

Este é um texto com uma palavra sobrescrita.

Tag <sub> Voltar

A tag <sub> é usada para definir um texto subscrito na página. O navegador mostra os textos dessa tag com uma fonte menor e abaixo da linha base. Por exemplo: <p>Este é um texto com uma <sub>palavra</sub> subscrita.</p> é um parágrafo com uma parte subscrita.

Este é um texto com uma palavra subscrita.

Combinando o uso de tags Voltar

As tags podem ser combinadas para criar efeitos diferentes nos textos. Por exemplo: <p>Este é um texto com uma <strong><i><u>palavra</u></i></strong> em negrito, itálico e sublinhado.</p> é um parágrafo com uma parte em negrito, itálico e sublinhado.

Este é um texto com uma palavra em negrito, itálico e sublinhado.


Listas Voltar

As listas são elementos que servem para organizar e apresentar uma série de itens na página. As listas podem ser de dois tipos: ordenadas e não ordenadas. As listas ordenadas são aquelas que têm uma ordem ou uma sequência lógica, e são representadas por números ou letras. As listas não ordenadas são aquelas que não têm uma ordem ou uma sequência lógica, e são representadas por símbolos ou marcadores.

Tipos e usos Voltar

As listas podem ser usadas para diversos fins, como mostrar uma lista de compras, uma lista de tarefas, uma lista de ingredientes, uma lista de passos etc. As listas podem ser criadas usando as seguintes tags:

  • <ol>: é usada para criar uma lista ordenada na página. Ela envolve os itens da lista, que são definidos pela tag <li>. Por exemplo: <ol><li>Primeiro item</li><li>Segundo item</li><li>Terceiro item</li></ol> é uma lista ordenada com três itens.

    1. Primeiro item
    2. Segundo item
    3. Terceiro item

  • <ul>: é usada para criar uma lista não ordenada na página. Ela envolve os itens da lista, que são definidos pela tag <li>. Por exemplo: <ul><li>Primeiro item</li><li>Segundo item</li><li>Terceiro item</li></ul> é uma lista não ordenada com três itens.

    • Primeiro item
    • Segundo item
    • Terceiro item

  • <li>: é usada para definir um item de uma lista na página. Ela fica dentro das tags <ol> ou <ul>. Por exemplo: <li>Primeiro item</li> é um item de uma lista.

Tag <ol> Voltar

A tag <ol> é usada para criar uma lista ordenada na página, ou seja, uma lista que tem uma ordem ou uma sequência lógica. A tag <ol> envolve os itens da lista, que são definidos pela tag <li>. Cada item da lista é representado por um número ou uma letra, que indica a sua posição na lista. Por exemplo: 

<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>
  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Esse código cria uma lista ordenada com três passos, que são mostrados na página com os números 1, 2 e 3. A lista ordenada pode ter diferentes tipos de numeração, como números romanos, números arábicos, letras maiúsculas, letras minúsculas etc. Para definir o tipo de numeração, podemos usar o atributo type na tag <ol>. Por exemplo: 

<ol type="I">
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>
  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Esse código cria uma lista ordenada com três passos, que são mostrados na página com os números romanos I, II e III. Alguns valores possíveis para o atributo type são:

  • 1: para números arábicos (1, 2, 3, …)
  • A: para letras maiúsculas (A, B, C, …)
  • a: para letras minúsculas (a, b, c, …)
  • I: para números romanos maiúsculos (I, II, III, …)
  • i: para números romanos minúsculos (i, ii, iii, …)

A lista ordenada também pode ter um valor inicial para a numeração, que pode ser diferente de 1. Para definir o valor inicial, podemos usar o atributo start na tag <ol>. Por exemplo: 

<ol start="5">
    <li>Quinto passo</li>
    <li>Sexto passo</li>
    <li>Sétimo passo</li>
</ol>
  1. Quinto passo
  2. Sexto passo
  3. Sétimo passo

Esse código cria uma lista ordenada com três passos, que são mostrados na página com os números 5, 6 e 7. O atributo start pode receber qualquer valor numérico, positivo ou negativo. Por exemplo: 

<ol start="-3">
    <li>Terceiro passo antes do zero</li>
    <li>Segundo passo antes do zero</li>
    <li>Primeiro passo antes do zero</li>
</ol>
  1. Terceiro passo antes do zero
  2. Segundo passo antes do zero
  3. Primeiro passo antes do zero

Esse código cria uma lista ordenada com três passos, que são mostrados na página com os números -3, -2 e -1.

Tag <ul> Voltar

A tag <ul>; é usada para criar uma lista não ordenada na página, ou seja, uma lista que não tem uma ordem ou uma sequência lógica. A tag <ul> envolve os itens da lista, que são definidos pela tag <li>. Cada item da lista é representado por um símbolo ou um marcador, que pode ser um círculo, um quadrado, um disco etc. Por exemplo: 

<ul>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ul>
  • Primeiro item
  • Segundo item
  • Terceiro item

Esse código cria uma lista não ordenada com três itens, que são mostrados na página com círculos. A lista não ordenada pode ter diferentes tipos de marcadores, que podem ser definidos usando o atributo type na tag <ul>. Por exemplo: 

<ul type="square">
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ul>
  • Primeiro item
  • Segundo item
  • Terceiro item

Esse código cria uma lista não ordenada com três itens, que são mostrados na página com quadrados. Alguns valores possíveis para o atributo type são:

  • disc: para discos (padrão)
  • circle: para círculos
  • square: para quadrados
  • none: para nenhum marcador

Tag <li> Voltar

A tag <li> é usada para definir um item de uma lista na página, seja ela ordenada ou não ordenada. A tag <li> fica dentro das tags <ol> ou <ul>, e pode conter qualquer tipo de conteúdo, como textos, imagens, links etc. Por exemplo: 

<ul>
    <li>Este é um texto</li>
    <li><img src="imagem.jpg"></li>
    <li><a href="https://www.dio.me">Este é um link</a></li>
</ul>
  • Este é um texto
  • Este é um link

Esse código cria uma lista não ordenada com três itens, que são um texto, uma imagem e um link. A tag <li> também pode ter um atributo chamado value, que serve para definir o valor do item na lista ordenada. Por exemplo: 

<ol>
    <li value="10">Décimo item</li>
    <li>Item seguinte</li>
    <li>Item seguinte</li>
</ol>
  1. Décimo item
  2. Item seguinte
  3. Item seguinte

Esse código cria uma lista ordenada com três itens, que são mostrados na página com os números 10, 11 e 12.

Lista dentro de lista Voltar

Uma lista pode conter outra lista dentro dela, criando uma lista aninhada ou uma sublista. Para criar uma lista dentro de outra lista, basta colocar a tag <ol> ou <ul> dentro da tag <li> da lista principal. Por exemplo: 

<ul>
    <li>Frutas
        <ul>
            <li>Banana</li>
            <li>Maçã</li>
            <li>Laranja</li>
        </ul>
    </li>
    <li>Legumes
        <ul>
            <li>Cenoura</li>
            <li>Alface</li>
            <li>Tomate</li>
        </ul>
    </li>
    <li>Verduras
        <ul>
            <li>Espinafre</li>
            <li>Couve</li>
            <li>Brócolis</li>
        </ul>
    </li>
</ul>
  • Frutas
    • Banana
    • Maçã
    • Laranja
  • Legumes
    • Cenoura
    • Alface
    • Tomate
  • Verduras
    • Espinafre
    • Couve
    • Brócolis

Esse código cria uma lista não ordenada com três itens, que são frutas, legumes e verduras. Cada item tem uma sublista não ordenada com três itens, que são exemplos de cada categoria. A lista aninhada pode ter um nível diferente da lista principal, ou seja, pode ser ordenada ou não ordenada, e pode ter um tipo diferente de numeração ou de marcador. Por exemplo: 

<ol>
    <li>Primeiro passo
        <ul>
            <li>Faça isso</li>
            <li>Faça aquilo</li>
            <li>Faça aquilo outro</li>
        </ul>
    </li>
    <li>Segundo passo
        <ol type="a">
            <li>Faça isso</li>
            <li>Faça aquilo</li>
            <li>Faça aquilo outro</li>
        </ol>
    </li>
    <li>Terceiro passo
        <ol type="I">
            <li>Faça isso</li>
            <li>Faça aquilo</li>
            <li>Faça aquilo outro</li>
        </ol>
    </li>
</ol>
  1. Primeiro passo
    • Faça isso
    • Faça aquilo
    • Faça aquilo outro
  2. Segundo passo
    1. Faça isso
    2. Faça aquilo
    3. Faça aquilo outro

Links Voltar

Os links são elementos que servem para conectar uma página HTML com outra página, um arquivo, um e-mail, um telefone etc. Os links permitem que o usuário navegue pela internet, acessando diferentes conteúdos e recursos. Os links podem ser criados usando a tag <a>, que é usada para definir um texto ou uma imagem como um link.

Tag <a> Voltar

A tag <a> é usada para definir um link na página. Ela tem um atributo chamado href, que indica o endereço do link. Por exemplo:
<a href="https://www.dio.me">DIO.me</a> é um link para o site da DIO.me. A tag <a> pode ter outros atributos, como target, title, download etc. Por exemplo: <a href="https://www.dio.me" target="_blank" title="Este é o site da DIO.me" >DIO.me</a> é um link para o site da DIO.me que será aberto em uma nova aba, que mostra o texto “Este é o site da DIO.me” quando o usuário passa o mouse sobre ele.

Atributo href Voltar

O atributo href é usado para definir o endereço de um link na página. Ele pode ter diferentes valores, dependendo do tipo de link que se quer criar. Alguns exemplos de valores são:

  • Um endereço absoluto: é um endereço completo, que inclui o protocolo, o domínio, o caminho e o arquivo. Por exemplo:
    <a href="https://www.dio.me">DIO.me</a> é um link para a página de pesquisa da DIO.me.

  • Um endereço relativo: é um endereço parcial, que se baseia na localização da página atual. Por exemplo:
    <a href="contato.html">Contato</a> é um link para a página “contato.html” que está na mesma pasta que a página atual.

  • Um endereço de e-mail: é um endereço que abre o programa de e-mail do usuário, com o destinatário já preenchido. Por exemplo:
    <a href="mailto:exemplo@exemplo.com">Enviar e-mail</a> é um link para enviar um e-mail para “exemplo@exemplo.com”.

  • Um número de telefone: é um número que abre o aplicativo de telefone do usuário, com o número já preenchido. Por exemplo:
    <a href="tel:+5511999999999">Ligar</a> é um link para ligar para “+5511999999999”.

  • Um identificador: é um nome que identifica uma parte de uma página, e que permite criar links internos na página. Para criar um identificador, podemos usar o atributo id em uma tag, e para criar um link para esse identificador, podemos usar o atributo href com o sinal de cerquilha (#) seguido do nome do identificador. Por exemplo: <h2 id="introducao">Introdução</h2> é um subtítulo com o identificador “introducao”, e <a href="#introducao">Ir para a introdução</a> é um link que leva para esse subtítulo.

  • target: é usado para definir o alvo de um link na página. Ele pode ser usado na tag <a>, e serve para indicar onde o link será aberto. Por exemplo: <a href="https://www.dio.me" target="_blank">DIO.me</a> é um link para o site da DIO.me que será aberto em uma nova aba. Alguns valores possíveis para o atributo target são:

    • _blank: para abrir o link em uma nova aba ou janela
    • _self: para abrir o link na mesma aba ou janela (padrão)
    • _parent: para abrir o link na aba ou janela pai, se houver
    • _top: para abrir o link na aba ou janela superior, se houver
    • nome: para abrir o link em uma aba ou janela com o nome especificado

  • title: é usado para definir o título de um elemento na página. Ele pode ser usado em várias tags, e serve para mostrar um texto quando o usuário passa o mouse sobre o elemento. Por exemplo: <a href="https://www.dio.me" title="Este é o site da DIO.me">DIO.me</a> é um link para o site da DIO.me que mostra o texto “Este é o site da DIO.me” quando o usuário passa o mouse sobre ele. O atributo title pode ser usado para dar uma descrição ou uma dica sobre o elemento, ou para melhorar a acessibilidade da página.

Outras tags Voltar

Além das tags que já foram mencionadas, existem outras tags que podem ser usadas para criar e formatar páginas HTML. Algumas dessas tags são:

  • <font>: é usada para definir a fonte de um texto na página. Ela pode ter atributos como face, size e color, que servem para definir o tipo, o tamanho e a cor da fonte. Por exemplo: <font face="Arial" size="4" color="green">Este é um texto verde com a fonte Arial</font> é um texto com a fonte Arial, tamanho 4 e cor verde.

    Este é um texto verde com a fonte Arial

  • <del>: é usada para definir um texto que foi deletado ou removido na página. O navegador mostra os textos dessa tag com uma linha no meio da fonte. Por exemplo: <p>Este é um texto com uma <del>palavra</del> deletada.</p> é um parágrafo com uma parte deletada.

    Este é um texto com uma palavra deletada.

  • <abbr>: é usada para definir uma abreviação ou um acrônimo na página. Ela pode ter um atributo chamado title, que serve para mostrar o significado da abreviação ou do acrônimo quando o usuário passa o mouse sobre o elemento. Por exemplo: <p>HTML é uma sigla para <abbr title="Linguagem de Marcação de Hipertexto">HyperText Markup Language</abbr>.</p> é um parágrafo que mostra o significado de HTML quando o usuário passa o mouse sobre ele.
